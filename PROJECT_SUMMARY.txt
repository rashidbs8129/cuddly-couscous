================================================================================
  API ENDPOINTS SUMMARY
================================================================================

AUTHENTICATION (PUBLIC)
  POST   /auth/send-otp        - Send OTP to phone
  POST   /auth/verify-otp      - Verify OTP, get token
  POST   /auth/sign-out        - Sign out

TAILOR MANAGEMENT (PROTECTED)
  POST   /tailors/profile      - Create tailor
  GET    /tailors/profile/:id  - Get tailor
  PUT    /tailors/profile/:id  - Update tailor
  DELETE /tailors/profile/:id  - Delete tailor
  GET    /tailors              - List all tailors
  GET    /tailors/stats/:id    - Get statistics

USER MANAGEMENT (PROTECTED)
  POST   /users                - Create user
  GET    /users/:id            - Get user
  PUT    /users/:id            - Update user
  DELETE /users/:id            - Delete user
  GET    /users                - List all users

MEASUREMENTS (PROTECTED)
  POST   /measurements         - Create measurement
  GET    /measurements/:id     - Get measurement
  GET    /measurements/user/:userId  - User measurements
  GET    /measurements/tailor/:tailorId - Tailor measurements
  PUT    /measurements/:id     - Update measurement
  DELETE /measurements/:id     - Delete measurement

================================================================================
  DATABASE SCHEMA
================================================================================

USERS TABLE
  id (UUID) - Primary key
  phone_number (VARCHAR UNIQUE) - Phone number
  name (VARCHAR) - Customer name
  email (VARCHAR) - Email address
  address (TEXT) - Physical address
  created_at (TIMESTAMP) - Creation time
  updated_at (TIMESTAMP) - Last update

TAILORS TABLE
  id (UUID) - Primary key
  name (VARCHAR) - Tailor name
  phone_number (VARCHAR UNIQUE) - Phone number
  email (VARCHAR) - Email address
  address (TEXT) - Shop address
  shop_name (VARCHAR) - Business name
  specialization (VARCHAR) - Specialty area
  created_at (TIMESTAMP) - Creation time
  updated_at (TIMESTAMP) - Last update

MEASUREMENTS TABLE
  id (UUID) - Primary key
  tailor_id (UUID FK) - Reference to tailor
  user_id (UUID FK) - Reference to user
  measurement_type (VARCHAR) - Type of measurement
  value (DECIMAL) - Measurement value
  unit (VARCHAR) - Unit (cm, inches, etc.)
  notes (TEXT) - Additional notes
  created_at (TIMESTAMP) - Creation time
  updated_at (TIMESTAMP) - Last update
