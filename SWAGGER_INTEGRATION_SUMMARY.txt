================================================================================
  QUICK TESTING WORKFLOW
================================================================================

1. START SERVER
   $ npm run start:dev
   
2. OPEN SWAGGER
   Visit: http://localhost:3000/api

3. AUTHENTICATE
   a) Click POST /auth/send-otp
   b) Click "Try it out"
   c) Enter: {"phoneNumber": "+919876543210"}
   d) Click "Execute"
   e) Check SMS for OTP

4. GET TOKEN
   a) Click POST /auth/verify-otp
   b) Click "Try it out"
   c) Enter phone and OTP (test: 123456)
   d) Copy access_token from response

5. AUTHORIZE
   a) Click "Authorize" button (top right)
   b) Paste: Bearer <your_access_token>
   c) Click "Authorize"

6. TEST ENDPOINTS
   a) Any endpoint (now has token)
   b) Click "Try it out"
   c) Enter required parameters
   d) Click "Execute"

================================================================================
  TROUBLESHOOTING
================================================================================

Issue: Swagger UI not loading
Solution: Ensure application is running on port 3000
         Check browser console for errors

Issue: Unauthorized errors on protected endpoints
Solution: Get token from /auth/verify-otp
         Click "Authorize" button
         Paste: Bearer <your_token>

Issue: Models not showing in Swagger
Solution: Ensure DTOs are properly exported
         Check @ApiProperty decorators are added
         Rebuild project: npm run build

Issue: Port 3000 already in use
Solution: Change PORT in .env file
         Restart application
         Access Swagger at new port
